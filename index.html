Algunos lenguajes de la bash

HTML+CSS+Javascript

  • HTML5

    • HTML5(Hyper Text Markup Language,versión 5) es la quinta revisión importante del lenguaje básico de la World Wide Web

    Usa la extensión.html

  • CSS 3

    • Hoja de estilo en cascada o CSS(siglas en inglés de casacading style sheets) es un lenguaje usado para definir y crear la presentación de un documento estructurado escrito en HTML

    Usa la extensión css

  • JavaScript

    • JavaScript(abreviando comúnmente "JS") es un lenguaje de programación utilizado principalmente en el lado del cliente (client-side),implementdo como parte de un navegador web permitiendo mejora en la interfaz de usuario y páginas web dinámicas

    Usa la extensión.js
